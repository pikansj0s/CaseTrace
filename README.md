# CaseTrace

**Free, automatic time tracking for Windows.** CaseTrace runs passively in the
background, captures everything you work on, and breaks the time down per case
and file — all locally on your own machine, with no account and no cloud.

**🌐 Website, screenshots & download → [casetrace.dk](https://casetrace.dk)**

**⬇️ Download:** [Microsoft Store](https://apps.microsoft.com/detail/9P96JCMB522V) · or the `.exe` installer from [Releases](https://github.com/pikansj0s/CaseTrace/releases/latest)

---

This repository hosts the **[casetrace.dk](https://casetrace.dk)** marketing site
(served via GitHub Pages) and the Windows installer (via GitHub Releases — not
committed to the tree).

- `index.html` — front page (Danish / English)
- `landing.css` — shared styles for the subpages
- Static site, no build step — open `index.html` or serve the folder.

CaseTrace is free software with a free license — for everyone who bills their
time: lawyers, architects, accountants, freelancers and more.
Learn more at **[casetrace.dk](https://casetrace.dk)**.
