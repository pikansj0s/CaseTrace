# CaseTrace

Marketing website for **CaseTrace** — automatic, passive time & folder
tracking for solo professionals (Windows desktop app).

- `index.html` — the full static site (inline CSS/JS, DA/EN toggle)
- `style.css` — shared styles
- No build step. Open `index.html` or serve the folder.

The Windows installer (`CaseTraceSetup_v4.exe`) is distributed via this
repo's **GitHub Releases**, not committed to the tree.

CaseTrace is free software with a free license.
